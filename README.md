🚀 Django Task Manager

A full-stack Task Management Web Application built using Django that allows users to register, log in, and manage their daily tasks efficiently with priorities and deadlines.

🔥 Features
🔐 User Authentication (Login / Register / Logout)
📝 Create, Update & Delete Tasks
📅 Set Due Dates for Tasks
⚡ Priority Levels (Low / Medium / High)
📋 Task Listing Dashboard
🎯 Clean and Minimal UI
